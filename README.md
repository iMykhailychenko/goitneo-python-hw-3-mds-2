### To run the project

```bash
pip install -r requirements.txt
python main.py
```

###  To run the tests
```bash
make test
```
or
```bash
python -m unittest discover tests
```


### To run linter
```bash
make lint
```
or 
```bash
python -m isort **/*.py
python -m black **/*.py
```
